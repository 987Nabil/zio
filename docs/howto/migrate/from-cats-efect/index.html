How to Migrate from Cats Effect to ZIO?

Cats IO[A] can be easily replaced with ZIO's Task[A] (an alias for ZIO[Any, Throwable, A]). Translation should be relatively straightfoward. Below, you'll find tables showing the ZIO equivalents of various cats.*'s methods.
